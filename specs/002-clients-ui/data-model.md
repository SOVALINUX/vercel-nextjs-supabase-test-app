### `public.clients`

| Column              | Type        | Nullable | Notes                                          |
| ------------------- | ----------- | -------- | ---------------------------------------------- |
| `id`                | uuid PK     | No       | `gen_random_uuid()`                            |
| `name`              | text        | No       |                                                |
| `link`              | text        | Yes      | URL; checked by `clients_link_format` constraint |
| `sales_executive_id`| uuid FK     | Yes      | → `public.users.id`                            |
| `account_manager_id`| uuid FK     | No       | → `public.users.id`                            |
| `_created_at`       | timestamptz | No       | `now()`                                        |
| `_created_by`       | uuid FK     | No       | → `auth.users.id`; set to `auth.uid()` on insert |
| `_updated_at`       | timestamptz | Yes      | auto-set by trigger                            |
| `_updated_by`       | uuid FK     | Yes      | → `auth.users.id`                              |
| `_deleted`          | boolean     | No       | `false`; soft-delete only                      |

**RLS Policies** (existing):

- `SELECT`: authenticated users can read `_deleted = false`
- `INSERT`: authenticated users; `_created_by = auth.uid()`
- `UPDATE`: `account_manager_id = auth.uid() OR _created_by = auth.uid()`
- `DELETE`: blocked

### `public.client_representatives`

| Column      | Type    | Nullable | Notes                                  |
| ----------- | ------- | -------- | -------------------------------------- |
| `client_id` | uuid FK | No       | → `public.clients.id` ON DELETE CASCADE |
| `user_id`   | uuid FK | No       | → `public.users.id` ON DELETE CASCADE  |
| PK          |         |          | `(client_id, user_id)`                 |

**RLS Policies** (existing):

- `SELECT`: authenticated users can read all
- `INSERT`: authenticated users; parent client's `account_manager_id = auth.uid() OR _created_by = auth.uid()`
- `DELETE`: same condition as INSERT
